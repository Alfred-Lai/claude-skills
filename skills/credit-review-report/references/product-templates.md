# 四种授信产品的差异化模板对比

## 对比总览

| 要素 | 传统授信 | 项目贷款 | 定增融资 | 集团授信 |
|------|---------|---------|---------|---------|
| 信用评审核心 | 经营周转能力 | 项目经济可行性 | 股票流动性和减持可行性 | 集团管控能力 |
| 第一还款来源 | 经营收入 | 项目自身现金流 | 减持+分红+薪酬 | 各成员经营收入 |
| 期限 | 1-3年 | 5-15年 | 1.5-3年 | 1-2年 |
| 利率趋势 | 3.0%-4.5% | 3.5%-5.0% | 打包价5%-7% | 3.2%-4.5% |
| 核心风险 | 经营波动/回款 | 完工风险/销售风险 | 股价下跌/减持受限 | 关联传导/过度融资 |

## 1. 传统授信业务

### 产品品种
短贷、中期流贷（每半年还本≥5%）、银承、国内信用证及项下融资、保函、商票融资、反向保理、银行承兑汇票贴现

### 关键分析段
- **流动资金需求测算**（必须）：营运资金量 - 自有资金 - 现有贷款 - 其他渠道
- **周转率分析**：应收账款周转、存货周转、应付账款周转
- **销贷比**：授信敞口 / 年营业收入，通常不超过40%

### 放款前提条件模板
```
1、办妥完整、合法、有效的用信手续，确保各项经营资质手续合法合规。
2、根据申请人实际经营需求合理安排用信额度，每笔业务须有明确资金用途，
   确保贸易背景真实，不得用于股权投资、固定资产投资等用途。
```

### 贷后管理要求模板
```
1、加强贷后管理，落实实地检查，跟踪日常经营管理和资金运营动态。
2、关注行业竞争、政策对经营的影响，重点关注应收款项、存货及回款变动。
3、关注融资变动、银行同业合作态度变动，动态调整信贷方案。
4、加强第一还款来源监控，直接销售回款占比不低于授信占比。
5、现有业务余额纳入本笔授信统一管理。
```

## 2. 项目贷款 / 房地产开发贷款

### 关键分析段
- **项目概况**：建设内容、总投资（需分项列示）、建设周期
- **资金拼盘**：自有资金比例（通常≥20%）、银行贷款、其他来源
- **经济测算**：达产收入/利润、销售价格假设、去化周期
- **还款测算**：按销售进度还款 或 固定分期还款

### 放款前提条件模板（房地产）
```
1、确保项目各项土地费用已交清，立项、环评、"四证"齐全合规有效。
2、项目自有资金比例不低于总投的XX%，先于我行贷款到位并投入使用。
3、借款合同写明：项目建设中若总投资扩大导致资金缺口，须追加资本金；
   未经我行书面同意，不得新增其他银行融资及对外担保；
   销售回笼款在本行账户归集，由本行监控。
```

### 贷后管理要求模板
```
1、严格按照相关管理办法执行，贷款资金根据实际工程进度发放。
2、投入资金（含自有资金、贷款、施工垫款）须与工程形象进度匹配。
3、切实做好贷后管理和销售回笼监控，销售回笼款按要求在本行归集。
4、动态关注区域调控政策变化，审慎评估对项目销售的影响。
```

## 3. 定增融资 / 股票收益权

### 结构特点
业务模式为"收益权模式"：合规资金→信托计划→融资人监管账户→定增募集账户
实质为股票收益权转让+远期回购，非真正意义上的股票质押贷款

### 关键分析段
- **减持新规分析**：是否受新规限制、减持节奏测算
- **质押分析**：质押率≤35%（限售股）/40%（流通股）、预警线160%、平仓线140%
- **股价下跌情景分析**：股价下跌10%/20%/30%/40%/50%时的减持股数和时间

### 业务方案模板要素
```
融资人：[姓名]，上市公司[职务]，持股[XX]股（[XX]%）
融资金额：不超过XX亿元
融资期限：不超过XX个月（须满足解禁日至少在到期日1个月前）
质押股数：XX万股（按20日均价测算，质押率XX%）
预警/平仓线：160%/140%
增信措施：质押+配偶连带责任保证
还款来源：(1)减持收入 (2)分红收入 (3)薪酬收入
```

### 风控要点
- 股价下跌30%时，是否能在2个季度内完成减持还款
- 减持后实控人地位是否变化
- 是否符合《上市公司股东减持股份管理暂行办法》

## 4. 集团授信

### 结构特点
先批集团总敞口，各成员主体/债项分报分审，额度在集团内统一管控

### 关键分析段
- **集团成员定位**：本部（管理/研发/贸易）+ 生产基地 + 贸易平台
- **资金管理模式**：统收统支 / 各自独立
- **集团整体融资**：各银行分布、品种、担保条件对比
- **管控要求**：哪些成员优先使用、启用前提

### 集团方案模板
```
拟同意给予[集团名称]集团主体授信额度人民币XX万元（敞口XX万元），有效期1年。
授信敞口由集团内成员使用，各成员主体、债项额度及业务余额均纳入集团主体额度统一管理。
新增成员单位主体授信或债项授信根据年度授权报批。
```

### 新增单一主体申请理由模板（超限额时使用）
```
(1)集团资质较好，风险较低。[具体说明]
(2)带动后续授信。以[流贷/其他]作为合作契机，营销后续[项目融资/其他]。
(3)维持综合效益。集团目前已带来[存款/结算/代发等具体数据]。
```
