# 行业分析联网搜索策略

## 搜索目标架构

每次行业分析需完成"四层次"信息收集：

### 宏观层：市场规模与趋势
搜索关键词模板：
- "[细分行业名称] 市场规模 [当前年份]"
- "[细分行业名称] 行业发展 趋势 预测"
- "[细分行业名称] 产业链 格局"
- "中国 [细分行业名称] 市场 增速"

### 中观层：产业链竞争格局
搜索关键词模板：
- "[细分行业名称] 竞争格局 市场份额"
- "[细分行业名称] 龙头企业 排名"
- "[上游原材料] 价格走势 [当前年份]"
- "[下游应用] 需求 增长 [当前年份]"

### 区域层：地方产业政策与环境
搜索关键词模板：
- "[城市/省份] 产业规划 [当前年份]"
- "[城市/省份] [行业] 产业集群"
- "[城市/省份] 政府工作报告 [当前年份] 产业"
- "[产业园区名称] 产业定位 政策"

### 微观层：可比公司对标
搜索关键词模板：
- "[可比上市公司名称] 年报 [年份] 营收 毛利率"
- "[行业名称] 上市公司 财务数据 对比"
- "[可比公司] 产能 产量 [年份]"

## 特定行业的搜索要点

### 新能源/锂电行业
- 搜索：锂电池产量、新能源车产销数据、负极材料/正极材料价格走势
- 关键下游：宁德时代/比亚迪/中创新航等电池厂出货量
- 区域关注：宁德/宜宾/常州/东莞的锂电产业集群

### 半导体/电子制造
- 搜索：全球半导体销售额、芯片产能、PCB/PCBA市场规模
- 关键下游：智能手机出货、服务器出货、汽车电子渗透率
- 区域关注：深圳/东莞/苏州/上海的电子制造集群

### 软件/信息技术
- 搜索：软件业收入增速、人工智能/大模型市场规模
- 关键下游：政府信息化预算、央企数字化采购
- 区域关注：北京/深圳/杭州/成都的软件产业集群

### 机械设备/自动化
- 搜索：工业机器人密度、自动化设备市场规模
- 关键下游：制造业固定资产投资增速、3C电子/汽车/新能源产能
- 区域关注：东莞/苏州/无锡的装备制造集群

### 房地产
- 搜索：当地房地产限购/限贷/限价政策、土地出让情况
- 关键数据：当地GDP、人口流入、住宅库存去化周期
- 区域关注：项目所在地的具体市场情况

## 数据来源优先级

1. 政府部门（统计局、工信部、发改委）——最高可信度
2. 行业协会（中国汽车工业协会、中国电子信息产业联合会等）——高可信度
3. 上市公司年报/招股书中的行业分析——较高可信度
4. 券商研报（中信、中金、华泰等头部券商）——中等可信度，交叉验证
5. 行业媒体/咨询机构（高工锂电、集邦咨询、IDC等）——参考性

## 在报告中的呈现原则

1. 所有行业数据必须标注来源和时间
2. 市场总量数据优先使用最新年份
3. 增速预测需标注预测区间或来源机构
4. 区域产业数据必须来自政府官方渠道
5. 避免使用过于绝对化的判断（"全球领先"→需要数据支撑）
