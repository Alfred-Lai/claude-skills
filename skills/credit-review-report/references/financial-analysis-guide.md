# 财务分析深度指南

## 常用财务指标计算公式

### 偿债能力指标
- 资产负债率 = 总负债 / 总资产 × 100%
- 流动比率 = 流动资产 / 流动负债
- 速动比率 = (流动资产 - 存货) / 流动负债
- 现金比率 = (货币资金 + 交易性金融资产) / 流动负债
- 有息负债比 = 本行授信敞口额度 / 企业有息负债
- 主营收入比 = 本行经营周转类授信敞口额度 / 企业上年主营收入

### 盈利能力指标
- 毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
- 净利率 = 净利润 / 营业收入 × 100%
- 销售利润率 = 营业利润 / 营业收入 × 100%
- 总资产收益率 ROA = 净利润 / 平均总资产 × 100%
- 净资产收益率 ROE = 净利润 / 平均净资产 × 100%

### 营运能力指标
- 应收账款周转率 = 营业收入 / 平均应收账款
- 应收账款周转天数 = 360 / 应收账款周转率
- 存货周转率 = 营业成本 / 平均存货
- 存货周转天数 = 360 / 存货周转率
- 应付账款周转率 = 营业成本 / 平均应付账款

### 现金流指标
- 经营现金流/净利润 = 经营活动净现金流 / 净利润
- 自由现金流 = 经营活动净现金流 - 资本性支出
- 现金收入比 = 销售商品提供劳务收到的现金 / 营业收入

## 指标趋势评判基准

### 资产负债率
- ≤30%：低负债，财务保守，融资空间大
- 30%-55%：适中，制造业合理区间
- 55%-70%：偏高，关注偿债安排
- ≥70%：高负债，需重点分析偿债能力（房地产和重资产行业可适当放宽至75%-80%）

### 流动比率
- ≥2：短期偿债充足
- 1.5-2：尚可
- 1-1.5：一般，需关注
- <1：短期偿债压力较大

### 速动比率
- ≥1：短期偿债较有保障
- 0.5-1：一般
- <0.5：存在短期偿债压力

### 经营现金流/净利润
- ≥1：利润含金量高
- 0.5-1：利润有一定含金量
- <0.5 或为负：关注应收/存货对资金占用

### 毛利率趋势
- 逐年提升：产品竞争力增强或成本控制改善
- 基本稳定：经营模式成熟
- 逐年下降：竞争加剧、原材料涨价或产品结构恶化

### 应收账款周转天数趋势
- 缩短：回款改善
- 稳定：经营正常
- 延长：关注下游回款能力和信用政策

## 财务报表质量评估

### 审计意见警示级别
- 标准无保留意见：信任度最高
- 带强调事项段的无保留意见：需关注强调事项的具体内容
- 保留意见：会计处理存在较大问题，需重点关注
- 无法表示意见/否定意见：报表不可信赖，原则上不宜介入

### 未审计报表的处理
- 注明"未审计"
- 通过纳税申报收入做交叉验证
- 关注纳税收入/报表收入比例（通常应在90%-110%之间，偏差过大需解释）

### 报表勾稽关系检查
- 资产负债表的未分配利润期末-期初 ≈ 利润表净利润 - 分红
- 现金流量表净额 ≈ 资产负债表货币资金期末-期初
- 营业收入 vs 销售商品提供劳务收到的现金（考虑应收/预收变动）

## 财务异动分析指引

对变动超过30%的科目进行分析时，需考虑：
1. 是业务规模扩张导致（营收增长→应收/应付/存货同步增长→正常）
2. 是经营策略变化导致（改变信用政策→应收增加→关注）
3. 是融资结构变化导致（短贷变长贷→有息负债结构变化）
4. 是非经常性事件导致（并购→商誉/长投增加）

## 专用术语映射

| 兴业报告用语 | 通用财务术语 |
|-------------|-------------|
| 毛利润 / 毛利润 | 毛利润 = 营业收入 - 营业成本 |
| 净利润率 | 净利率 = 净利润 / 营业收入 |
| 经营创现能力 | 经营活动现金流质量 |
| 直接销售回款占比 | 本行结算量 / 企业营业收入 |
| 综合效益 | EVA + VN + 存款 + 结算的综合评价 |
| 销贷比 | 授信敞口 / 年营业收入 |
